# Loaders Modernized

A Space Aged, compact loader the size of a single tile.  These loaders should be relatively UPS friendly
and do not use LUA script to move items.  They also do not use hidden inserters to move inventory.  However, they
do use power.  So keep those power poles handy!

## Features

- Works in Factorio 2.0 and supports Space Age.
- Control by circuit supported.
- Works with cargo wagons.
- Loaders snap to belts and away from non-belts.
- Stack items on to the belts based on belt stacking technology level.
- Split Lanes!!!
- Wait for Full Stack configurable per loader.
- Respect insert limits for machines configurable per loader.
- AAI Loader graphics can be used as long as AAI Loader mod is installed.
- Stacked loaders can be used without Space Age!  This requires the use of the [Stack Inserters](https://mods.factorio.com/mod/stack-inserters) mod.
- Factorio 1.x maps with a [Miniloaders](https://mods.factorio.com/mod/miniloader) can be migrated using a companion mod. - [Miniloader migration](https://mods.factorio.com/mod/miniloader-migration).

## Chutes

This mod does add cheep chutes to the game.  The default behavior limits them to input (belt -> inventory) only, and they completely (over)fill machines.
There are settings that can be toggled to allow output, enable filtering, and cause them to respect insert limits of machines.

## Alternate belt sets supported

An API is available so that overhaul and belt mods can directly add loaders.
[Loaders Modernized: Integrations](https://mods.factorio.com/mod/loaders-modernized-integrations) provides support for the following mods:

- Krastorio 2
- Space Exploration
- Bob's Logistics
- Pyanodons
- 5Dim's New Transport
- Ultimate Belts Space Age
- Extended Vanilla: Modular Logistics & Robotics
- Iper Belt
- Wooden Logistics
